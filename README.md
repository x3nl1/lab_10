\## Запуск проекта



\### 1. Локальный запуск



\*\*Запуск Go сервиса:\*\*



&#x20;   cd m\_task\_api

&#x20;   go mod tidy

&#x20;   go run .



\*\*Запуск Python тестов (в отдельном терминале):\*\*



&#x20;   cd python\_client

&#x20;   pip install requests pytest

&#x20;   pytest



\### 2. Запуск через Docker



&#x20;   docker-compose up --build

\## Особенности реализации



\- Валидация через struct tags (binding) в Gin  

\- Обработка всех ошибок в Go (каждый error проверяется)  

\- Передача данных между сервисами через JSON  

\- JWT используется для защиты endpoint `/protected`  

\- Python клиент взаимодействует с Go API через HTTP  

\- Реализованы unit-тесты для Go и Python
